software required.

## What's in the workbook

| Tab | What's inside |
|-----|---------------|
| IR Incident Log | Record every incident from detection to closure with severity and status tracking |
| IR Playbook | Step-by-step actions across all six response phases with owners and timing |
| Severity Matrix | Classify every incident as Critical, High, Medium, or Low with response SLAs |
| RACI Chart | Who is Responsible, Accountable, Consulted, and Informed for every activity |
| Tabletop Exercise | A ransomware scenario with six facilitator injects — ready to run with your team |
| Post-Mortem Template | Complete within 5 days of every incident closure |
| Action Tracker | Every remediation action with owner, priority, due date, and status |

---

## The six phases

| Phase | Description |
|-------|-------------|
| 1 — Preparation | Assign roles, define severity levels, test backups, maintain asset inventory |
| 2 — Detection | Triage alerts, assign incident ID, determine scope, classify severity |
| 3 — Containment | Isolate systems, preserve evidence, block threat vector |
| 4 — Eradication | Find root cause, remove threat, reset credentials, verify clean |
| 5 — Recovery | Restore from backup, validate operations, assess regulatory obligations |
| 6 — Post-incident review | Run post-mortem, track actions, update the plan |

---

## How to use the tabletop exercise

The tabletop tab contains a ransomware scenario with six facilitator injects.
Run it with your IR team in about two hours — no preparation needed.

1. Read the scenario overview aloud
2. Work through each inject in order
3. Record team decisions and gaps in the Notes column
4. Add gaps to the Action Tracker after the exercise
